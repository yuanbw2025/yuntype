# 如何高效阅读一本书
阅读是一种终身学习的方式。掌握正确的阅读方法，能让你事半功倍。
## 一、选书：找到值得读的书
不是所有书都值得从头读到尾。学会筛选是高效阅读的第一步。
- 看目录，判断结构是否清晰
- 看序言，了解作者的写作动机
- 看评价，但不要被评分绑架
> 一本好书，是浓缩的思考。
## 二、阅读：主动阅读四步法
带着问题去读比无目的翻页效率高出十倍。
1. 略读：花 15 分钟快速翻阅全书框架
2. 精读：带着问题逐段阅读核心章节
3. 笔记：用自己的话重新表述关键观点
4. 回顾：一周后复盘加深记忆
> 阅读的目的不是记住，而是理解。
