# 云中书 / YunType — 开发分期计划

> 本文档定义开发里程碑、每期交付内容和验收标准。

---

## 总体规划

| 阶段 | 周期 | 核心交付 | 对应功能 |
|------|------|---------|---------|
| **Phase 1: MVP** | 2周 | 公众号排版核心流程 | F1+F2+F5+F7+F8 |
| **Phase 2: AI** | 1周 | AI分析+微调面板 | F3+F4+F6 |

## Phase 1: MVP（公众号排版核心）

**目标**: 粘贴文章 → 随机推荐排版 → 预览 → 复制富文本到微信

### 交付清单

- [ ] 项目脚手架（Vite + React + TypeScript）
- [ ] 11套配色方案数据（colors.ts）
- [ ] 5种排版结构数据（layouts.ts）
- [ ] 4种装饰风格数据 + HTML模板（decorations.ts）
- [ ] 3种字体气质数据（typography.ts）
- [ ] 原子组合引擎（atoms/index.ts）
- [ ] Markdown 解析器（render/markdown.ts）
- [ ] 公众号 HTML 渲染器（render/wechat.ts）— 全部内联CSS，微信兼容
- [ ] 文章输入组件（ArticleInput）
- [ ] 实时预览组件（WechatPreview）— 375px 模拟
- [ ] "换一个"按钮 — 随机切换原子组合
- [ ] 复制富文本功能（export/clipboard.ts）
- [ ] 导出 HTML 功能（export/download.ts）
- [ ] 底部状态栏（字数、当前方案名称）
- [ ] Vercel 部署

### 验收标准

1. 粘贴一篇 Markdown 文章，点击"换一个"切换不同排版风格
2. 预览区实时显示排版效果
3. "复制富文本"后粘贴到微信公众号编辑器，样式完整保留
4. 660种组合中随机抽样10种，全部渲染正确

---

## Phase 2: AI 分析 + 微调

**目标**: AI 自动推荐排版 + 滑条微调

### 交付清单

- [ ] API 配置弹窗（ApiConfig 组件）
- [ ] 统一 AI 客户端（ai/client.ts）— 支持9家提供商
- [ ] 文章分析模块（ai/analyzer.ts）— Prompt + JSON解析
- [ ] 微调面板（StylePanel）— 5根滑条
- [ ] 滑条 → CSS参数映射逻辑
- [ ] "恢复AI推荐"按钮
- [ ] localStorage 存储 API 配置
- [ ] 无 API Key 时的优雅降级

### 验收标准

1. 配置千问/DeepSeek API Key后，粘贴文章点击"AI分析"，3-10秒内返回推荐方案
2. 推荐方案的配色/排版/装饰/字体与文章气质匹配
3. 拖动滑条，预览实时更新（<200ms延迟）
4. 不配置API Key，所有核心功能正常可用

---

## Phase 3: 小红书模式

**目标**: 输入文章 → 生成图片组 → 下载 PNG/ZIP

### 交付清单

- [ ] 平台切换组件（PlatformSwitch）
- [ ] 小红书分页算法（自动分页）
- [ ] AI辅助分页（可选，需API）
- [ ] 封面图模板（5种变体）
- [ ] 内容图模板（5种变体）
- [ ] 尾图模板
- [ ] 字体加载管理器（fonts.ts）— 16种免费字体
- [ ] 小红书 HTML 渲染器（render/xiaohongshu.ts）
- [ ] 小红书预览组件（XhsPreview）
- [ ] html2canvas 集成 — HTML→PNG
- [ ] 单张下载 + ZIP打包下载
- [ ] 图片尺寸选择（3:4 / 1:1 / 16:9）

### 验收标准

1. 粘贴一篇1500字文章，自动分为6-10张图片
2. 封面图有大标题，尾图有引导关注
3. 自定义字体（如得意黑、霞鹜文楷）在图片中正确显示
4. 导出的PNG尺寸为1080×1440（3:4），文字清晰
