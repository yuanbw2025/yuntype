# 云中书公众号 HTML 插件 — 使用说明

本插件配合 **云中书（YunType）** 排版工具使用，让你在微信公众号编辑器中一键插入精美的 HTML 排版内容。

---

## 第一步：安装插件（只需做一次）

### Chrome 浏览器

1. 打开 Chrome，地址栏输入 `chrome://extensions/` 并回车
2. 右上角开启 **开发者模式**（Developer mode）
3. 点击 **加载已解压的扩展程序**（Load unpacked）
4. 选择本文件夹（即包含 `manifest.json` 的这个文件夹）
5. 插件安装完成 ✅

### Edge 浏览器

1. 打开 Edge，地址栏输入 `edge://extensions/` 并回车
2. 左下角开启 **开发人员模式**
3. 点击 **加载解压缩的扩展**
4. 选择本文件夹
5. 插件安装完成 ✅

---

## 第二步：用云中书生成排版 HTML

1. 打开云中书排版工具：**[云中书 YunType](https://yuntype.pages.dev)**
2. 在左侧输入 Markdown 内容，选择配色、字体、比例
3. 点击右上角 **导出 HTML** 按钮，复制生成的 HTML 代码

---

## 第三步：插入到微信公众号编辑器

1. 用安装了插件的浏览器打开 **微信公众号后台** → 新建图文
2. 进入编辑器页面后，右侧会出现一个 **紫色悬浮图标**（`< >`）
3. 点击该图标，弹出插入面板
4. 将复制好的 HTML 代码粘贴进去
5. 点击 **插入** 按钮
6. 排版格式即可直接加入到编辑器中 🎉

---

## 效果说明

- 支持云中书生成的所有排版样式（配色、字体、多栏布局等）
- 插入后在编辑器中可见，保存发布后读者端同样生效
- 不影响公众号原有内容，可在现有文章基础上追加

---

## 常见问题

**Q：插件图标没有出现？**
A：刷新公众号编辑器页面，等待 2-3 秒让插件初始化完成。

**Q：插入后样式消失了？**
A：微信公众号会过滤部分 CSS 属性，建议使用云中书生成的内联样式版本，兼容性更好。

**Q：每次打开浏览器都要重新启用？**
A：Chrome/Edge 在开发者模式下加载的插件不会自动禁用，正常情况下一次安装长期有效。
