# ☁️ 云中书 / YunType

> AI 驱动的内容排版工具 — 粘贴文章，一键生成公众号排版和小红书图片组。

---

## 这是什么？

**云中书** 帮助公众号和小红书创作者，不需要设计能力，也能做出专业、好看、有品牌辨识度的排版。

```
粘贴文章 → AI 推荐排版方案 → 微调 → 复制到微信 / 下载图片组
```

### 核心特性

- 🎨 **660种排版组合** — 11配色 × 5排版 × 4装饰 × 3字体，原子化组合不撞款
- 🤖 **AI 自动推荐** — 分析文章内容和文风，推荐最匹配的排版方案
- 🎚️ **滑条微调** — 调"感觉"而不是调参数（色温、装饰、间距...）
- 📝 **公众号模式** — 输出微信兼容的内联CSS富文本，直接粘贴
- 📸 **小红书模式** — 自动分页生成图片组，支持16种免费商用字体
- 🔑 **用户自带API Key** — 零后端，隐私安全，核心功能离线可用

## 技术栈

| 层 | 技术 |
|---|------|
| 框架 | React + TypeScript |
| 构建 | Vite |
| 样式 | Tailwind CSS（工具UI）+ 纯内联CSS（排版输出） |
| 图片生成 | html2canvas |
| 字体 | Google Fonts + jsDelivr CDN |
| 部署 | Vercel |

